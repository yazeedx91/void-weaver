## 📋 **DEPLOYMENT INSTRUCTIONS**

### **1. Environment Setup**

```bash
# Frontend (Next.js 15)
cd frontend
npm install
npm run dev    # Port 3000

# Backend (Al-Hakim Server)  
cd backend
pip install -r requirements.txt
pip install -r agent_requirements.txt
python al_hakim_server.py    # Port 8001
```

### **2. Environment Variables**

```env
# Core API Keys
OPENAI_API_KEY=your_key_here
TAVILY_API_KEY=your_key_here
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_key

# Localization
DEFAULT_LOCALE=ar
SAUDI_TIMEZONE=Asia/Riyadh
CURRENCY_CODE=SAR
```

### **3. Database Setup**

- Run SQL migration in Supabase
- Enable pgvector extension
- Set up Arabic text search

### **4. Launch Sequence**

```bash
# Terminal 1: Backend
python al_hakim_server.py

# Terminal 2: Frontend  
npm run dev

# Access Points:
# English: http://localhost:3000/en
# Arabic: http://localhost:3000/ar (Default for KSA IPs)
```
